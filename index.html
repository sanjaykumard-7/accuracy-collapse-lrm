About the Project

This project investigates Accuracy Collapse ‚Äî a phenomenon where Large Reasoning Models (LRMs) show decreasing accuracy as the logical or arithmetic complexity of tasks increases, despite sufficient model capacity. By simulating reasoning tasks and testing open-weight models, this study aims to uncover how reasoning performance can be improved through structured mitigation techniques.
